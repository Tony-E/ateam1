Program Codes

Program Codes are assigned to each observer at those observatories where different people or teams are working. A program code is added (by the MPC) to each observation in order to show which observer or team submitted that observation. The codes shown below are those assigned to current and past Slooh Members. When a program code is greater than 9, it is shown as a special character in Obs80-format observations.
